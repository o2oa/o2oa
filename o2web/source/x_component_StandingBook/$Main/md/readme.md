# 欢迎使用台账
* 台账应用实现了对流程数据和内容管理数据的过滤、展现和导出。
* 台账应用支持同时对多个流程应用和内容管理的台账。
* 台账应用支持台账列的添加、删除、排序和拖动等功能。
* 台账应用支持配置常用台账条目，在常用台账条目中可选择流程应用或内容管理应用，并配置过滤及展现的字段。
![image.png](../x_component_StandingBook/$Main/md/img/2022-11-22_15-27-33.png)

## 权限
* 普通用户可以看到参与流转的流程数据和具有编辑及阅读权限的内容管理数据。
* 流程应用管理员可以看到相关应用的所有流程数据。
* 内容管理栏目管理员可以看到相关栏目的所有内容管理数据。
* 管理员（Manager角色）可以看到所有的流程数据和内容管理数据。

## 过滤
系统提供了几种过滤方式:
* 默认过滤，鼠标移到过滤条件上，可展现过滤区域，每个过滤项都包含值和数量（括号中），点击可进行过滤。
* 文本字段，文本字段需要输入完整的值进行匹配。输入完成后回车可执行过滤。 
* 日期字段，本应用对日期字段提供日期选择组件进行过滤，过滤的日期将被限定在所有值的最小值和最大值之间。 
* 数字字段，本应用对数字字段提供数字范围进行过滤，过滤的数字将被限定在所有值的最小值和最大值之间。输入完成后回车可执行过滤。 

## 展现
* 台账应用每页显示50条数据。 
* 台账应用默认按创建时间倒序排列。
* 台账应用支持字符串字段、日期字段和数字字段的点击排序。
* 台账应用支持调整列，可以添加或者删除展现的列，或者调整列的顺序。

## 导出
点击界面上的导出按钮，可以根据过滤条件和展现列导出当前应用的数据，导出形式为Excel。
