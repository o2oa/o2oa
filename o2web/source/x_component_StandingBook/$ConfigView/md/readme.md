
# 配置
用户点击左导航的下方可以配置常用条目

## 条目配置项
台账条目配置包含的功能如下，启用或禁用台账、设置台账的标题、设置台账的流程应用和内容管理应用；设置过滤条件的字段，设置默认展现的字段，设置调整列的候选字段。<br />具体设置项如下表：

| **设置项** | **说明** |
| --- | --- |
| 启用/禁用 | 禁用后在使用界面的导航中不再展现 |
| 台账名称 | 显示在使用界面导航中的文字 |
| 流程应用 | 点击后可多选，本条目使用的流程数据。<br />流程应用管理员可以看到相关应用。<br />管理员（Manager角色）可以看到所有流程应用。 |
| 内容管理 | 点击后可多选，本条目使用的内容管理数据。<br />内容管理栏目管理员可以看到相关栏目。<br />管理员（Manager角色）可以看到所有的内容管理栏目。 |
| 台账字段 | 具体看后面的台账字段配置说明 |
| 可见范围 | 默认为空。空则全员可见。点击可选择个人、组织、群组，选择后范围内的用户才可以查看。 |

## 台账字段配置
台账字段根据选择的流程应用及内容管理列出。台账字段配置项如下表：

| **设置项** | **说明** |
| --- | --- |
| 字段名称 | 不可修改，包括三类字段，分别是系统默认过滤字段，系统默认展现字段、设计人员增加的业务字段。 |
| 字段文本 | 输入框，使用界面的过滤条件显示的文本和表格标题文本。 |
| 可过滤 | 选中后，可以在使用界面的过滤条件中出现。<br />新建条目时，系统默认过滤条件默认选中。<br />系统默认展现字段不能过滤。<br />业务字段中只有字符串类型，数字类型和日期类型可以过滤。 |
| 可展现 | 选中后，可以在使用界面“调整列”操作的选项中出现。<br />新建条目时，系统默认展现字段（列）默认选中。<br />系统默认过滤字段不能展现。 |
| 默认展现 | 选中后，系统打开该台账时默认展现该列（字段）；但如果用户进行了“调整列”操作，以用户调整的列为准。 |
